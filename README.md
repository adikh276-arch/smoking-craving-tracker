# Smoke Craving Tracker

A comprehensive application to track smoking cravings and help you quit smoking.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- i18next for Internationalization

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Access the application at `http://localhost:8080/smoke_craving_tracker/` (after configuration)

## Deployment

The application is containerized using Docker and served via Nginx.
A CI/CD pipeline is set up using GitHub Actions to build and push the Docker image.
